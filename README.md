# Benjidotsh Tap

## How do I install these formulae?

`brew install benjidotsh/tap/<formula>`

Or `brew tap benjidotsh/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "benjidotsh/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
